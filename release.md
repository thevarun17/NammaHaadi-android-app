# Namma Haadi (Our Path) 🚀

**Release v1.0.0-alpha: Initial Launch**

We are excited to announce the first release of **Namma Haadi (Our Path)**!  
This version introduces the core community-driven features designed to make our roads safer and more transparent.

---

## ✨ Key Features
- 📢 **Real-time Reporting**: Report potholes, flooding, road work, or blockages with photos and GPS locations.
- 📍 **Interactive Live Map**: View all community-reported hazards on a live Google Map to plan safer routes.
- 🔔 **Urgent Broadcasts**: Send and receive instant "Danger," "Warning," and "Info" alerts to keep the neighborhood informed.
- 🔄 **Status Tracking**: Monitor the progress of reports as they move from Active → In-Progress → Resolved.
- 🏆 **Leaderboard & Gamification**: Earn points and unlock "Road Guardian" badges for your contributions.
- 📱 **Secure Login**: Simple and fast Phone OTP-based authentication.

---

## 🛠️ Technical Highlights
- Built entirely with **Jetpack Compose** for a modern, fluid UI.
- Real-time data synchronization powered by **Firebase Firestore**.
- Dependency injection using **Hilt**.

## 📦 How to Install
1. Download the `app-debug.apk` attached in the release.
2. Open the file on your Android device.
3. Allow *Install from Unknown Sources* if prompted.
4. Start reporting!
